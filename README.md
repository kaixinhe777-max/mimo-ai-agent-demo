> **让超长文档与复杂代码不再成为效率瓶颈**  
> 一个专为个人开发与学术学习设计的 AI Agent，实现文档精读、代码生成与审查、多任务迭代优化的全链路自动化。

---

## 🎯 项目背景

在个人开发与学术研究中，我们常常面临三大核心痛点：

- 📄 **超长文档阅读慢**：动辄几十页的论文、技术手册、API 文档，难以快速定位关键信息。  
- 🐞 **代码调试反复踩坑**：同样的错误多次出现，缺乏系统的代码审查与测试流程。  
- ⏰ **多任务处理效率低**：阅读、写作、编码、实验交替进行，注意力碎片化。

**LongDoc-CodeAgent** 应运而生，通过大模型驱动的长链推理与多轮迭代优化，将上述痛点转化为高效工作流。

---

## ✨ 核心功能

| 模块 | 功能描述 |
|------|----------|
| 📖 长文本智能解析 | 自动分块处理超长文档，保留语义完整性，支持 PDF、Markdown、TXT 等格式 |
| 🔍 核心信息提取 | 抽取文档中的关键论点、数据、公式、结论，生成结构化摘要 |
| 🧠 需求理解与规划 | 根据用户自然语言描述，拆解任务步骤，生成执行计划 |
| 💻 代码生成与逐行审查 | 生成 Python/Shell 等代码，并进行逐行注释、风险提示与最佳实践建议 |
| ✅ 单元测试自动编写 | 为生成的代码自动构造测试用例，验证逻辑正确性 |
| 📊 结果总结与输出 | 输出包含执行结果、测试报告、改进建议的完整报告 |
| 🔁 多轮迭代优化 | 根据反馈自动修正方案，支持连续对话式改进 |

---

## 🏗️ 技术架构

- **框架**：LangChain（链式调用、工具集成、记忆管理）
- **语言模型**：GPT-4 / MiMo-V2.5（计划接入）
- **核心机制**：
  - 长文档分块 + 递归摘要
  - 代码生成专用 Prompt Chain
  - 单元测试自动生成器
  - 基于对话状态的多轮迭代记忆
- **工具集成**：Python REPL、文件 I/O、终端命令执行（受控）
